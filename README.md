# Portfólio - Davi Vinicius

Este é o repositório do meu portfólio pessoal, desenvolvido para apresentar meus projetos, experiências e habilidades como Engenheiro de Software.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias modernas focadas em performance e experiência de usuário:

- **React:** Biblioteca JavaScript para construção da interface de usuário.
- **Vite:** Ferramenta de build extremamente rápida para o ecossistema frontend.
- **Tailwind CSS:** Framework CSS utilitário para estilização rápida e responsiva.
- **Framer Motion:** Biblioteca de animações em React para criar transições fluidas.
- **React Router:** Para o roteamento entre as páginas (Home, Projetos, Sobre, Experiência, Contato).
- **Lucide React:** Biblioteca de ícones bonitos e leves.
- **Context API:** Para gerenciamento do estado global da aplicação (como Tema Claro/Escuro e Idiomas).

## ✨ Funcionalidades

- **Design Responsivo:** A interface se adapta perfeitamente a dispositivos móveis, tablets e monitores desktop.
- **Temas Customizáveis:** Suporte a Tema Claro, Tema Escuro e Tema Roxo (Dark Purple).
- **Multi-idioma:** Suporte integrado a traduções (Português do Brasil `pt-BR` e Inglês `en-US`).
- **Navegação Fluida:** Rotas do React gerenciadas pelo `react-router-dom`, com transições animadas.

## 📦 Como instalar e executar o projeto localmente

Siga os passos abaixo para rodar o projeto na sua máquina:

### Pré-requisitos
- É necessário ter o **Node.js** e o **npm** (ou **pnpm** / **yarn**) instalados na máquina.

### Passos

1. Clone o repositório para a sua máquina:
```bash
git clone https://github.com/davi1710231/Portfolio_Davi.git
```

2. Entre no diretório do projeto: (Caso seja um monorepo, acesse a pasta da aplicação web)
```bash
cd Portfolio_Davi
cd apps/web # Se aplicável
```

3. Instale as dependências:
```bash
npm install
# ou
yarn install
# ou
pnpm install
```

4. Execute o servidor de desenvolvimento:
```bash
npm run dev
# ou
yarn dev
# ou
pnpm run dev
```

5. O servidor será inicializado e você poderá acessar a aplicação através do navegador, geralmente no endereço: `http://localhost:5173`.
